# 인허가·보험·소방 심화 (on-demand)

SKILL.md §2(행정 컴플라이언스)의 상세판. **해당 행사에 컴플라이언스 판단이 필요할 때만** 펼쳐 본다. 구체 기준은 지자체·관할마다 다르므로, 여기 적힌 건 **무엇을 확인해야 하는지의 체크 축**이지 법적 확정이 아니다 — 확정이 필요하면 관할에 확인하거나 사용자에게 확인 요청(`ask_user_question`).

## 1. 인허가 (행사 성격별)
- **옥외·공원·광장 사용**: 시설 사용허가/사용료. 공원·광장은 별도 점용 신청.
- **도로 점용·통제**: 차로·인도 일부 사용 시 도로점용허가, 교통 통제는 경찰 협의.
- **집회·옥외 모임**: 규모·성격에 따라 집회 신고.
- **음향·소음**: 야간·주거지 인접 시 음향 사용 허가/소음 기준.
- **주류 제공**: 판매면 주류 관련 허가, 무료 제공도 장소 정책 확인.
- **임시 구조물**(무대·천막): 규모 따라 가설 구조물 신고·안전 점검.

판단 순서: 행사 성격(옥외/규모/주류/구조물) → 해당 축만 골라 관할 확인 → state에 "확인 완료/확인 불가" 기록.

## 2. 보험
- **행사배상책임보험(기본)**: 참가자·제3자 신체·재산 사고 대비. 옥외·고위험·다수 인원이면 사실상 필수.
- **보장 범위 체크**: 1인 한도·총 한도, 음식(식중독)·구조물·자원봉사자 포함 여부.
- **업체 보험 확인**: 케이터링·AV·시공 업체의 자체 배상책임보험 가입 여부(계약서에 명시 요구).
- **예산 연결**: 보험료는 예산의 예비비/인력·운영 축에서 확보 → [[budget-policy]].

## 3. 소방·안전
- **수용 인원 한도**: 장소 허용 인원 초과 금지. 좌석/입석 구분.
- **피난 동선**: 비상구 확보·미차단, 통로 폭, 안내 표지.
- **소화 설비**: 소화기 위치·수량, 필요 시 옥내소화전 점검.
- **신고 대상**: 일정 규모 이상·실내 다중이용 행사는 소방 신고/사전 협의 대상일 수 있음 → 관할 확인.
- **전기·화기**: 임시 전원 용량·과부하, 화기 사용(푸드트럭·촛불) 시 별도 안전 조치.

## 4. 식품 위생
- **케이터링/푸드 업체**: 영업신고·위생등급 확인. 현장 조리면 가열·보관 온도 관리.
- **무료 시식·나눔**도 위생 책임은 남는다.
- 알레르기·종교 식이는 SKILL.md §5 참조.

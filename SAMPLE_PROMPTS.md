# 샘플 프롬프트 (테스트·데모용)

작업공간 AI 에이전트 입력창에 **그대로 붙여넣어** 쓰는 기획 브리프 모음.
매번 새로 짤 필요 없이 골라 쓰면 됩니다. 각 프롬프트 아래 `→`는 그게 **자극하는 기능**(어디를 보면 되는지).

> 형식: `"행사명" 행사를 기획해줘.` + `조건:` 줄들. 행사명만 바꿔도 됨.
> 비용 팁: 빠르게 보려면 인원·예산을 작게. 풀 파이프라인(researcher 병렬·writer⇄critic)은 토큰이 좀 든다.

---

## 기업·일반 행사

### 1) 세미나·컨퍼런스 (기본)
```
"2026 프론트엔드 컨퍼런스" 행사를 기획해줘.
조건:
인원: 200명
장소: 코엑스나 비슷한 컨퍼런스 홀
예산: 2000만원
날짜: 9월 중 평일
```
→ 예산 배분(세미나·컨퍼런스 비율)·장소 조사·체크리스트·제안서 기본 흐름.

### 2) 네트워킹·리셉션
```
"파트너사 네트워킹 리셉션" 행사를 기획해줘.
조건:
인원: 120명
장소: 강남권 호텔 연회장
예산: 3000만원
날짜: 10월 첫째 주 저녁
```
→ 케이터링 비중 큰 배분(네트워킹·리셉션), 격식 톤 메일(notice-writer).

### 3) 사내 워크숍 (소규모)
```
"디자인팀 워크숍" 행사를 기획해줘.
조건:
인원: 30명
장소: 양양 근처 워크숍 공간
예산: 자유
날짜: 다음 달 중
```
→ 워크숍·소규모 배분, 예산 "자유" 처리(에이전트가 적정 예산 제안).

### 4) 대형 야외 페스티벌 (날씨·리스크 자극)
```
"가을 뮤직 페스티벌" 행사를 기획해줘.
조건:
인원: 1000명
장소: 한강공원 야외
예산: 8000만원
날짜: 9월 마지막 주 토요일
```
→ **옥외 → 날씨(get_weather)·우천 리스크·예비비 상향**(risk-assessment), 대형 행사 배분.

## 학생회 행사

### 5) 새내기 배움터 (새터)
```
"컴퓨터공학과 새터" 행사를 기획해줘.
조건:
인원: 150명
장소: 대성리 MT 시설 (1박 2일)
예산: 학생회비 + 학교 지원금 600만원
날짜: 8월 말
```
→ 새터 배분(숙박·교통·안전↑), **미성년 음주·이동/야간 안전**(risk-assessment 학생 심화).

### 6) 과 MT
```
"전자공학과 가을 MT" 행사를 기획해줘.
조건:
인원: 60명
장소: 가평 펜션
예산: 300만원
날짜: 10월 중 주말
```
→ MT 배분, 이동·숙박 안전 점검.

### 7) 대학 축제 주점
```
"공대 대동제 학과 주점" 행사를 기획해줘.
조건:
인원: 부스 운영 200명 규모
장소: 캠퍼스 대운동장 부스
예산: 250만원
날짜: 9월 축제 기간
```
→ 축제·주점 배분(식음료·주류·무대), **주류·부스 화기·미성년**(risk-assessment), 학교 시설 승인.

### 8) 과 체육대회
```
"기계공학과 체육대회" 행사를 기획해줘.
조건:
인원: 100명
장소: 학교 운동장
예산: 150만원
날짜: 10월 중 금요일
```
→ 체육대회 배분(물품·상품·안전·의무), 캠퍼스 시설.

## 기능 콕 집어 자극

### 9) 공휴일/연휴 충돌 (캘린더 MCP)
```
"추석 직전 고객 감사 행사" 행사를 기획해줘.
조건:
인원: 80명
장소: 서울 시내 컨퍼런스룸
예산: 1000만원
날짜: 9월 넷째 주
```
→ **추석 연휴 인접 → 캘린더 MCP로 공휴일·요일 따져 날짜 조정 제안**(ask_user_question).

### 10) 예산 명시 + 재기획 유도
```
"개발자 밋업" 행사를 기획해줘.
조건:
인원: 200명
장소: 성수동 이벤트홀 (야외 가능)
예산: 500만원
날짜: 10월 중
```
→ 빠듯한 예산 → critic이 비현실성 지적 → **사용자 판단(예산 재배분/축소)** 게이트. 야외라 날씨도.
